Module Description

This module is an academic development system. It allows user to propose courses to different faculties. The module allows people to review each course, after reviewing the user can then decide whether to accept or reject the course proposed.

Key features

The Buddies module is implemented generically:-

  1. Course are listed according to faculties in the main page.

  2. Users can delete, review or edit each course proposal.

  3. Clicking each course link takes the user to the history of different revisions of the course and review thereof.

  4. Clicking each revision takes the user to the document itself.
