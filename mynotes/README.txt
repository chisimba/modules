To quickly see how the module works, install it, then go to resources/tmp.

In there are some sql files that you can run in MySql and the data will be
inserted into your tables. And you can quickly get a look at how my notes 
modules functions. There will be a tag cloud on the left and a couple of notes
on the right with paging. The latest two notes will show with html stripped
and with words count less than 200.
